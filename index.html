Simple CSS Selectors

CSS Selectors allows you to select HTML elements tht you want ot style

There are five selector categories: Simple Selectors, Combinator Selectors, Pseudo-class Selectors, Pseudo-element Selectors and Attribute Selectors
Selectors Selector Example Description
#id firstname selects the element with id="firstname"
.class .header .header selects all elements with class="header"
element p All the <p> elements get selected
element,elements,.. div, p All the <div> elements and all the <p> elements gets selected
* * Selects all the elements

CSS Specificity

Priority Example Description
Id Selectors <h1 style="color: pink;"> Highest priority, directly applied with the style attribute
Id selectors #navbar Second highest priority, identified by the unique id attribute of an element
Classes and pseudo-classes .test, :hover Third highest priority, targeted using class names
Attributes [type="text"] Low priority, applies to attributes
Elements and pseudo-elements h1, ::before, ::after Lowest priority, applies to HTML elements and pseudo-elements
